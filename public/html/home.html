Administración de la API #MásDatos

Instrucciones:

Para evitar posibles errores, es recomendable realizar los siguientes pasos:

  1. Descargar el archivo con las palabras mediante el botón "Descargar palabras"
  2. Abrir dicho archivo en Excel u otro intérprete de archivos CSV
  3. Realizar las modificaciones pertinentes
  4. Guardarlo sobreescribiendo el mismo archivo
  5. Subir el nuevo listado de palabras mediante "Subir palabras" adjuntando el archivo modificado
